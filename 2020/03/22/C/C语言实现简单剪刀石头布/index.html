关键表达式理解

(man - computer + 4) % 3 - 1

  • man与computer的取值只有0， 1， 2，man与computer对比后的结果只有三种，所以我们可以确定 %3 是存在表达式中的
  • man与computer对比后，man赢的值是1， -2，平的值是0，man输的值是-1，2。
  • 因为我们需要得到：赢的值 % 3 = 固定的值，平的值 % 3 = 固定的值， 输的值 % 3 = 固定的值
  • 加3： man赢的值是4， 1，平的值是3，man输的值是2，5
  • % 3： man赢的值是1，平的值是0，man输的值是2
  • 这样子已经可以去判定了，但不是很顺眼
  • 加4： man赢的值是5，2，平的值是4，man输的值是3，6
  • %3： man赢的值是2，平的值是1，man输的值是0
  • 这样子顺眼多了，但再减1会更顺眼
  • 最后就可以得到 ： (man - computer + 4) % 3 - 1
